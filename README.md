# Landing Page — Teste Técnico Front-end (Concentrix)

Este projeto foi desenvolvido como parte do **teste técnico para a vaga de Desenvolvedor Front-end** na **Concentrix**.  
O objetivo foi construir uma **landing page moderna, responsiva e performática**, seguindo boas práticas de desenvolvimento web e utilizando tecnologias atuais do ecossistema React/Next.js.

---

## Tecnologias Utilizadas

- **Next.js 16 (App Router)**
- **React**
- **TypeScript**
- **Sass/SCSS** para estilização
- **Arquitetura de componentes**
- **ESLint + Prettier** para padronização e qualidade de código

---

## Objetivo do Projeto

Criar uma página que:

- possua **layout otimizado**, limpo e profissional  
- seja **100% responsiva** (mobile-first)  
- apresente **bom desempenho** e carregamento rápido  
- utilize componentes reaproveitáveis  
- respeite boas práticas de acessibilidade  


O foco principal é demonstrar **capacidade técnica**, **organização**, **clareza de código** e **boas práticas de front-end**.

## Como Executar o Projeto

### 1. Instalar dependências

Abrir o projeto no terminal e rodar os comandos abaixo

npm install
# ou
yarn install

npm run dev

o Projeto será executado na porta a seguir

http://localhost:3000

É possível visualizar o projeto em funcionamento no endereço abaixo:
